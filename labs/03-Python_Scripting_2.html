Setup
  1. In your Linux home directory, make a new directory called lab03.
  2. Copy Drosophila.csv and sequence2.txt from the /export/data/657.019/text directory to your lab03 directory.
  3. Copy these Python script files from the /export/data/657.019/python directory to your lab03 directory:
    • calc_aa_percent.py
    • check_conds_genes.py
    • translate_dna_to_protein.py
Calculating amino acid percentages
  1. We want to define a function that takes two arguments, a protein sequence and a 1-letter amino acid residue code, and then calculates the percentage of the sequence where the amino acid appears.
  2. We also want to define another function that takes a protein sequence and a list of amino acid codes as arguments, and then calculates the percentage of the sequence where the amino acids in the list appear.
  3. The calc_aa_percent.py file contains a partially done Python script that defines the two functions and tests them. Complete the TO DO sections.
  4. Make sure all the test function calls return correct values, as percentages (not ratios).
Checking conditions for genes
  1. The Drosophila.csv file is a comma-separated file about some made-up genes of Drosophila. Each line has this information about a gene: species, sequence, gene name, and expression level.
  2. The check_conds_genes.py file contains a partially done Python script to print, for each gene, the following things on one line separated by comma:
    • Gene name
    • A message saying whether the GC percentage is high (above 55%), normal (between 45% and 55%), or low (below 45%), followed by the percentage itself
    • A message saying whether the gene length is normal (defined as being between 90 and 110, inclusive), followed by the length itself, only when the length is normal. Otherwise the message is empty.
    • A message saying the gene name starts with K or H, only when this is true. Otherwise the message is empty.
  3. Check the script output, to confirm that your gene condition tests are correct.
Translating DNA into protein
  1. Given a genetic code, we want to translate DNA sequences to protein sequences.
  2. The translate_dna_to_protein.py file contains a partially done Python script to define a function to do the translation and test the function. Complete the TO DO sections, noting that this script does three main things:
    • Define the genetic code as a dictionary variable.
    • Define a function to translate a DNA sequence to a protein sequence.
    • Test translation with several DNA sequences.
  3. Make sure your script translates the test DNA sequences correctly.
